class phypno.graphoelement.Spindles[source]¶

Bases: object

Class containing all the spindles in one dataset.

chan_name¶

ndarray (dtype=’U’) – list of channels

mean¶

ndarray (dtype=’float’) – mean of each channel

std¶

ndarray (dtype=’float’) – standard deviation of each channel

det_value¶

ndarray (dtype=’float’) – value used for detection for each channel

sel_value¶

ndarray (dtype=’float’) – value used for selection for each channel

spindle¶

list of dict –

list of spindles, where each spindle contains:
  • start_time : float
    start time of the spindle
  • end_time : float
    end time of the spindle
  • peak_time : float
    time of the highest value
  • peak_val : float
    the highest value
  • chan’: str
    channel label
